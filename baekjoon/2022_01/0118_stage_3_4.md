### #15552 빠른 A+B
```python
import sys
num = int(input())

# input()보다 빠른 입력인 sys.stdin.readline().split() 활용

    A, B = map(int,sys.stdin.readline().split())
```

## 4단계 While문
### #10952 A+B - 5
```python
import sys

while 1:
    A, B = map(int, sys.stdin.readline().split())
    if 0:
        break
    print(A + B)
```

### #10951 A+B - 4
```python
import sys
# 파일이 끝까지 가면 EOFError를 발생시킨다. 에러가 나면 멈추도록 try, except를 사용한다.
while 1:
    try:
        A, B = map(int, sys.stdin.readline().split())
        print(A + B)
    except:
        break
```

### #1110 더하기 사이클
```python
import sys

num = int(input())
num_first = num
cnt = 0

while 1:
    if num < 10:
        a = 0
        b = num
    else:
        a = num // 10
        b = num % 10
    c = a + b
    num = b*10 + c%10
    cnt += 1
    if num == num_first:
        break

print(cnt)
```
