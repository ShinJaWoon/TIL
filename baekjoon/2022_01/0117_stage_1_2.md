## 2단계 if문
### #1330 두 수 비교하기
```python
a, b = map(int, input().split(' '))
if a > b:
    print('>')
elif a < b:
    print('<')
else:
    print('==')
```

### #14681 사분면 고르기
```python
x = int(input())
y = int(input())

if x > 0:
    if y > 0:
        print(1)
    else:
        print(4)
else:
    if y > 0:
        print(2)
    else:
        print(3)
```

### #2884 알람 시계
```python
H, M = map(int, input().split(' '))
M = M - 45
if M < 0:
    H = H - 1
    M = M + 60

if H < 0:
    H = H + 24
    
print(f'{H} {M}')
```
