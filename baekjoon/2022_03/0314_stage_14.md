## 14단계 백트래킹
### #15649 N과 M (1)
https://www.acmicpc.net/problem/15649
풀이
```python
N, M = map(int, input().split())

numbers = list(range(1, N+1))
visit = [0]*(1+N)
res = [0]*M
def f(x, M):
    if x == M:
        print(*res)
    else:
        for i in range(1, N+1):
            if visit[i] == 0:
                visit[i] = 1
                res[x] = i
                f(x+1, M)
                res[x] = 0
                visit[i] = 0
            
f(0, M)
```
