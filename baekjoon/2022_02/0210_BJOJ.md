### #2628 종이자르기
문제 설명
```markdown
직사각형 모양의 종이가 있다. 이 종이는 가로방향과 세로 방향으로 1㎝마다 점선이 그어져 있다. 가로 점선은 위에서 아래로 1번부터 차례로 번호가 붙어 있고, 세로 점선은 왼쪽에서 오른쪽으로 번호가 붙어 있다.
입력으로 종이의 가로 세로 길이, 그리고 잘라야할 점선들이 주어질 때, 가장 큰 종이 조각의 넓이가 몇 ㎠인지를 구하는 프로그램을 작성하시오.
```

입력
```markdown
첫줄에는 종이의 가로와 세로의 길이가 차례로 자연수로 주어진다. 가로와 세로의 길이는 최대 100㎝이다. 
둘째 줄에는 칼로 잘라야하는 점선의 개수
셋째 줄부터 마지막 줄까지 한 줄에 점선이 하나씩 아래와 같은 방법으로 입력된다. 
가로로 자르는 점선은 0과 점선 번호가 차례로 주어지고, 세로로 자르는 점선은 1과 점선 번호가 주어진다. 입력되는 두 숫자 사이에는 빈 칸이 하나씩 있다.
```

출력
```markdown
첫째 줄에 가장 큰 종이 조각의 넓이를 출력한다. 단, 넓이의 단위는 출력하지 않는다.
```


풀이
```python
def area(box):
    result = 0
    for i in range(box[0], box[2]):
        for j in range(box[1], box[3]):
            result += 1
    return result


y_max, x_max = map(int, input().split())
num_cutting = int(input())
# 리스트
boxes = []
area_boxes = []
x_value = [0, x_max]
y_value = [0, y_max]
for cutting in range(num_cutting):
    x_or_y, number = map(int, input().split())

    # 세로선
    if x_or_y:
        y_value.append(number)
    # 가로선
    else:
        x_value.append(number)

x_value.sort()
y_value.sort()

# 한칸 한칸마다 넓이를 부여하는 방식
for i in range(len(x_value)-1):
    for j in range(len(y_value)-1):
        # box들은 왼쪽위 모서리와 오른쪽 아래 모서리로 정한다.
        # 왼쪽 위 모서리에 1칸 부여
        boxes.append([x_value[i], y_value[j], x_value[i+1], y_value[j+1]])

for box in boxes:
    area_boxes.append(area(box))
    

# 결과 출력
print(max(area_boxes))
```

y_max, x_max = map(int, input().split())

boxes = []

    # 세로선
    if x_or_y:
        y_value.append(number)
    # 가로선
    else:
        x_value.append(number)

x_value.sort()
y_value.sort()

# 가로와 세로선만 있기 때문에 가장 큰 사각형은 가장 큰 변들을 가지고 있다.
line_x = []
line_y = []

for i in range(1, len(x_value)):
    line_x.append(x_value[i] - x_value[i-1])
for i in range(1, len(y_value)):
    line_y.append(y_value[i] - y_value[i-1])
    
# 결과 출력
print(max(line_x) * max(line_y))
