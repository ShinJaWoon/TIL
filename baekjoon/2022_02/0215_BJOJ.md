## 0215 IM 대비 문제풀기
### #
문제 설명
```markdown
25개의 칸으로 이루어진 빙고판에 1부터 25까지 자연수
사회자가 몇 번째 수를 부른 후 철수가 "빙고"를 외치게 되는지
```

입력
```markdown
첫째 줄부터 다섯째 줄까지 빙고판에 쓰여진 수가 가장 위 가로줄부터 차례대로 한 줄에 다섯 개씩 빈 칸을 사이에 두고 주어진다. 
여섯째 줄부터 열째 줄까지 사회자가 부르는 수가 차례대로 한 줄에 다섯 개씩 빈 칸을 사이에 두고 주어진다. 
빙고판에 쓰여진 수와 사회자가 부르는 수는 각각 1부터 25까지의 수가 한 번씩 사용
```

출력
```markdown
첫째 줄에 사회자가 몇 번째 수를 부른 후 철수가 "빙고"를 외치게 되는지 출력한다.
```

풀이
```python
# 빙고판을 2차원 리스트에 저장
board = [list(map(int, input().split())) for r in range(5)]
# 사회자가 부르는 번호를 1차원 리스트에 저장
MC = []
for _ in range(5):
    MC.extend(list(map(int, input().split())))

# 빙고 개수 체크 함수. 0이 사회자가 부른 숫자
def check_bingo(board):
    bingo = 0
    # 가로줄 검색
    bingo += board.count([0]*5)
    # 세로줄 검색. 전치 행렬을 통해 [0, 0, 0, 0, 0]을 찾는다.
    board_T = list(map(list, (zip(*board))))
    bingo += board_T.count([0]*5)
    # 대각선 검색
    # 왼쪽 위에서 오른쪽 아래로
    for i in range(5):
        if board[i][i] != 0:
            break
    else:
        bingo += 1
    # 오른쪽 위에서 왼쪽 아래로
    for i in range(5):
        if board[i][4-i] != 0:
            break
    else:
        bingo += 1
    # 빙고 횟수가 3 이상이면 True 아니면 False
    if bingo >= 3:
        return True
    else:
        return False

cnt = 0
# 사회자가 번호를 부르면 맞는 번호를 찾아 0으로 바꾼다.
for num in MC:
    cnt += 1
    for r in range(5):
        for c in range(5):
            if board[r][c] == num:
                board[r][c] = 0
    # 빙고가 3개 이상되면 카운트를 출력하고 break
    if check_bingo(board):
        print(cnt)
        break
```
