### #2669 직사각형 네개의 합집합의 면적 구하기
문제 설명
```markdown
평면에 네 개의 직사각형이 놓여 있는데 그 밑변은 모두 가로축에 평행하다. 이 네 개의 직사각형들은 서로 떨어져 있을 수도 있고, 겹쳐 있을 수도 있고, 하나가 다른 하나를 포함할 수도 있으며, 변이나 꼭짓점이 겹칠 수도 있다.
```

입력
```markdown
입력은 네 줄이며, 각 줄은 직사각형의 위치를 나타내는 네 개의 정수로 주어진다. 첫 번째와 두 번째의 정수는 사각형의 왼쪽 아래 꼭짓점의 x좌표, y좌표이고 세 번째와 네 번째의 정수는 사각형의 오른쪽 위 꼭짓점의 x좌표, y좌표이다. 모든 x좌표와 y좌표는 1이상이고 100이하인 정수이다.
```

출력
```markdown
첫 줄에 네개의 직사각형이 차지하는 면적을 출력한다.
```

풀이
```python
boxes = []
for i in range(4):
    boxes.append(list(map(int, input().split())))

# 1X1 사각형을 기준으로 왼쪽 아래 모서리에 넓이 1을 부과하여 1칸으로 생각한다.
# dots는 set으로 선언하여 중복된 점을 방지한다.
dots = set()
for box in boxes:
    for x in range(box[0], box[2]):
        for y in range(box[1], box[3]):
            dots.add((x, y))
print(len(dots))
```

### #2635 수 이어가기
문제 설명
```markdown
1. 첫 번째 수로 양의 정수가 주어진다.
2. 두 번째 수는 양의 정수 중에서 하나를 선택한다.
3. 세 번째부터 이후에 나오는 모든 수는 앞의 앞의 수에서 앞의 수를 빼서 만든다. 예를 들어, 세 번째 수는 첫 번째 수에서 두 번째 수를 뺀 것이고, 네 번째 수는 두 번째 수에서 세 번째 수를 뺀 것이다.
4. 음의 정수가 만들어지면, 이 음의 정수를 버리고 더 이상 수를 만들지 않는다.
```

입력
```markdown
첫 번째 수가 주어진다. 이 수는 30,000 보다 같거나 작은 양의 정수이다.
```

출력
```markdown
첫 번째 줄에는 입력된 첫 번째 수로 시작하여 위의 규칙에 따라 만들 수 있는 수들의 최대 개수를 출력한다.

둘째 줄에 그 최대 개수의 수들을 차례대로 출력한다. 이들 수 사이에는 빈칸을 하나씩 둔다.
```

풀이
```python
N = int(input())
cnt_max = 0   # 최대 개수
result = []   # 최대 개수일 대 수열 리스트

# 1~N 까지 i를 넣고 반복문을 돌린다.
for i in range(1, N+1):
    # 기본 수열과 초기 횟수=2
    arr = [N, i]
    cnt = 2
    N1 = i
    N2 = N - N1
    # 음의 정수가 될 때까지 반복문을 돌린다.
    while N2 >= 0:
        arr.append(N2)
        N2, N1 = N1 - N2, N2
        cnt += 1
    # 최대 횟수와 최대 횟수일 때 수열 리스트 저장
    if cnt >= cnt_max:
        cnt_max = cnt
        result = arr[:]

print(cnt_max)
print(*result)

```

### #1244 스위치 켜고 끄기
문제 설명
```markdown
‘1’은 스위치가 켜져 있음을, ‘0’은 꺼져 있음을 나타낸다.
남학생은 스위치 번호가 자기가 받은 수의 배수이면, 그 스위치의 상태를 바꾼다.
여학생은 자기가 받은 수와 같은 번호가 붙은 스위치를 중심으로 좌우가 대칭이면서 가장 많은 스위치를 포함하는 구간을 찾아서, 그 구간에 속한 스위치의 상태를 모두 바꾼다. 이때 구간에 속한 스위치 개수는 항상 홀수가 된다.
```

입력
```markdown
첫째 줄에는 스위치 개수가 주어진다. 스위치 개수는 100 이하인 양의 정수
둘째 줄에는 각 스위치의 상태가 주어진다. 켜져 있으면 1, 꺼져있으면 0이라고 표시하고 사이에 빈칸이 하나씩 있다.
셋째 줄에는 학생수가 주어진다. 학생수는 100 이하인 양의 정수  
넷째 줄부터 마지막 줄까지 한 줄에 한 학생의 성별, 학생이 받은 수
남학생은 1로, 여학생은 2로 표시하고, 학생이 받은 수는 스위치 개수 이하인 양의 정수이다. 학생의 성별과 받은 수 사이에 빈칸이 하나씩 있다.
```

출력
```markdown
스위치의 상태를 1번 스위치에서 시작하여 마지막 스위치까지 한 줄에 20개씩 출력한다. 예를 들어 21번 스위치가 있다면 이 스위치의 상태는 둘째 줄 맨 앞에 출력한다. 켜진 스위치는 1, 꺼진 스위치는 0으로 표시하고, 스위치 상태 사이에 빈칸을 하나씩 둔다.
```

풀이
```python
num_switch = int(input())
switch = list(map(int, input().split()))
num_stu = int(input())
for stu in range(num_stu):
    gender, num = map(int, input().split())

    # 남학생
    if gender == 1:
        # 받은 num의 배수의 스위치들을 바꾼다.
        for i in range(num-1, num_switch, num):
            switch[i] = int(not bool(switch[i]))
        # print(switch)

    # 여학생
    if gender == 2:
        # 받은 num을 기준으로 대칭이 되는 최대 구역을 찾는다.
        # 대칭 구역의 한쪽의 최대 길이를 구한다.
        min_range = min(num_switch - num, num-1)
        # print(f'min_range={min_range}')
        # 대칭이 아닌 경우 break를 걸고 직전의 idx를 저장한다.
        # 중심은 항상 바꾼다.
        switch[num-1] = int(not bool(switch[num-1]))
        for idx in range(min_range+1):
            if switch[num-1-idx] == switch[num-1+idx]:
                switch[num-1+idx] = int(not bool(switch[num-1+idx]))
                switch[num-1-idx] = int(not bool(switch[num-1-idx]))
            else:
                break

            

# print(switch)
# 20개씩 끊어서 출력
indexfrom = 0
while num_switch > 20:
    print(*switch[indexfrom:indexfrom+20])
    indexfrom += 20
    num_switch -= 20
print(*switch[indexfrom:])
```
