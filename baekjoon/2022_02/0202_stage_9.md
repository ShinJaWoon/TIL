## 9단계 기본 수학 2
### #4153 직각삼각형
https://www.acmicpc.net/problem/4153
```python
import sys

while True:
    x, y ,z = map(int, sys.stdin.readline().split())
    if x == y == z == 0:
        break
    m = max(x, y, z)
    if x**2 + y**2 + z**2 - 2* m**2 == 0:
        print('right')
    else:
        print('wrong')
```
