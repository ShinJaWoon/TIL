### #1018 체스판 다시 칠하기
문제 설명
```markdown
M×N 크기의 보드, 정사각형은 검은색으로 칠해져 있고, 나머지는 흰색
잘라서 8×8 크기의 체스판
체스판을 색칠하는 경우는 두 가지뿐이다. 하나는 맨 왼쪽 위 칸이 흰색인 경우, 하나는 검은색인 경우
 8×8 크기의 체스판으로 잘라낸 후에 몇 개의 정사각형을 다시 칠해야겠다고 생각
 다시 칠해야 하는 정사각형의 최소 개수를 구하는 프로그램
```

입력
```markdown
첫째 줄에 N과 M이 주어진다. N과 M은 8보다 크거나 같고, 50보다 작거나 같은 자연수이다. 둘째 줄부터 N개의 줄에는 보드의 각 행의 상태가 주어진다. B는 검은색이며, W는 흰색이다.
```

출력
```markdown
첫째 줄에 지민이가 다시 칠해야 하는 정사각형 개수의 최솟값을 출력한다.
```

풀이
```python
import sys

N, M = map(int, sys.stdin.readline().split())
board = list()

for row in range(N):
    board.append(sys.stdin.readline().strip())

# 칠하는 횟수를 저장하는 리스트
brush = []

for row in range(0, N-7):
    for col in range(0, M-7):
        board_str = ''
        for row8 in range(8):
            # 한줄은 그대로, 한줄은 리버스로 받음 (지그재그를 검사하기 위함)
            if row8 % 2:
                board_str += board[row+row8][col:col+8][::-1]
            else:
                board_str += board[row+row8][col:col+8]

        # 첫번째가 검정, 두번째가 흰색
        cnt= 0
        for b in range(0, 64, 2):
            if board_str[b] != 'B':
                cnt += 1
            if board_str[b+1] != 'W':
                cnt += 1
        brush.append(cnt)
        # 첫번째가 흰색인 경우는 첫번째가 검정인 경우에서 틀린 경우만 색칠하면 됨
        brush.append(64-cnt)

print(min(brush))
```

### #1436 영화감독 숌
문제 설명
```markdown
종말의 숫자란 어떤 수에 6이 적어도 3개이상 연속으로 들어가는 수
제일 작은 종말의 숫자는 666이고, 그 다음으로 큰 수는 1666, 2666, 3666
N번째 종말의 숫자?
```

입력
```markdown
첫째 줄에 숫자 N이 주어진다. N은 10,000보다 작거나 같은 자연수이다.
```

출력
```markdown
첫째 줄에 N번째 영화의 제목에 들어간 수를 출력한다.
```

풀이
```python
N = int(input())
n = 1
N_1 = N+1
d = dict()
for i in range(0, N):
    if n == N+1:
        break
    i_str = str(i)
    i_len = len(i_str)

    i_str_re = i_str[::-1]
    if i_str[-3:] == '666':
        for k in range(0, 10**(3)):
            d.update({n: f'{i_str}{k:0>3d}'})
            n += 1
        continue
    elif i_str_re[:2] == '66':
        for k in range(0, 10**(2)):
            d.update({n: f'{i_str}6{k:0>2d}'})
            n += 1
        continue
    elif i_str_re[:1] == '6':
        for k in range(0, 10):
            d.update({n: i_str+'66'+str(k)})
            n += 1
        continue
    else:
        d.update({n: i_str+'666'})
        n += 1
title = int(d[N])
print(title)
```
